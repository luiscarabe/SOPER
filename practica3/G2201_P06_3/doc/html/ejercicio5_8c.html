Function Documentation

int ejercicio2Sem ( int  num)

Esta funcion (del ejercicio 2) crea una zona de memoria compartida, posteriormente, realiza un fork. Desde el proceso hijo se solicita la informacion de los nuevos clientes por pantalla. Desde el proceso padre, se imprime desde el manejador esta informacion cuando se recibe la senial SIGUSR1 que envia el hijo una vez recibida la informacion. En este ejercicio lo implementamos con semáforos.
